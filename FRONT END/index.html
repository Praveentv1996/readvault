BookVault
v1.0  ·  Personal Reading Tracker

BookVault is a personal desktop app to catalogue and track every book, audiobook, e-book and video-book you read — all backed by a local PostgreSQL database and surfaced through a clean, fast UI.

Tech Stack
LayerTechnology
Backend APIPython · Flask · Flask-CORS
DatabasePostgreSQL (psycopg2)
FrontendVanilla HTML / CSS / JavaScript
Desktop Shellpywebview (native window)
AI IntegrationPostgreSQL MCP Server (Claude Code)
Features
  • Add, edit & delete books with a modal form
  • Multi-field filter: Author · Source · Status · Featuring
  • Global search across title, author & featuring
  • Sortable columns with single-click toggle
  • Live stats — total, authors, finished, source breakdown
  • Auto-detects book type from source (AudioBook / E-Book / VideoBook / Book)
  • Paginated table with configurable rows-per-page
  • White & Grey themes, persisted via localStorage
  • F5 / refresh button reload without restart
  • MCP server for AI-assisted queries from Claude Code
